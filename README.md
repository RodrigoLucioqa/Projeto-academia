# HardCore - Academia de Crossfit

Bem-vindo ao repositório oficial da **HardCore**, uma academia de Crossfit fictícia. Este projeto é um site institucional desenvolvido para simular a apresentação de uma academia, seus planos, instalações e permitir que os usuários entrem em contato.

## Visão Geral

O site da HardCore foi desenvolvido para oferecer uma experiência moderna e responsiva, destacando os serviços oferecidos pela academia. Ele inclui:

- Páginas de apresentação: Informações sobre a academia, sua missão e valores.
- Planos de assinatura: Diferentes opções de planos para atender às necessidades dos alunos.
- Formulário de contato: Para que os usuários possam entrar em contato com a equipe da HardCore.
- Design responsivo: O site é totalmente responsivo, funcionando em dispositivos móveis, tablets e desktops.

## Tecnologias Utilizadas

- HTML5: Estruturação do conteúdo do site.
- CSS3: Estilização e design responsivo.
- JavaScript: Interatividade e funcionalidades dinâmicas.
- SwiperJS: Biblioteca para criar sliders/carrosséis interativos.
- Google Fonts: Utilização da fonte Teko para um design moderno.
- Google Maps: Integração de mapa para localização da academia.
